YAML::XS - Support "booleans"

  • You can now exchange data with JSON modules, and booleans will survive
  • Set $YAML::XS::Boolean
use YAML::XS; $YAML::XS::Boolean = 'JSON::PP'; # use JSON::PP::Boolean objects $YAML::XS::Boolean = 'boolean'; # use boolean.pm objects my $data = Load("some setting: true");
